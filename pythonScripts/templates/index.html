Welcome to the dissolution camera system

The dissolution camera system uses a set of Raspberry Pi cameras (version 2) attached to Raspberry Pi 0 W controllers. Each controller is running a HTTP server that provides an API to control the camera and provides a stream to view previous time lapse recordings.
